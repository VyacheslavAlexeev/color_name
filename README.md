# ColorName
ColorName is a simple ruby gem that allows you to get a name of any color.

## Install

```ruby
gem install color_name
```

Or add the following line to your Gemfile:
```ruby
gem 'color_name'
```

## Usage

```ruby
# You have to
ColorName.get('#f00') # Red
ColorName.get('#e02253') # Amaranth
```
